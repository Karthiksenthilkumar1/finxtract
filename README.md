# Finxtract AI: Enterprise PDF Intelligence

Finxtract is a professional-grade financial document processor designed to automate the extraction of critical data from Balance Sheets and Profit & Loss statements. Utilizing neural mapping and advanced layout analysis, Finxtract transforms complex financial PDFs into structured, actionable intelligence.

## 🚀 Key Features

- **Autonomous Neural Extraction**: Advanced parsing of complex financial document layers.
- **Enterprise-Grade Dashboard**: Real-time monitoring of your document processing pipeline.
- **Secure Data Synthesis**: Localized processing for high-security financial datasets.
- **XLSX Intelligence Export**: Seamlessly transition from raw document to structured analysis.
- **AI-Powered Executive Analysis**: Strategic "Virtual CFO" summaries of extracted financial data.
- **Financial Performance Visualization**: Interactive trend analysis of Revenue and Profit across documents.

## 🛠 Tech Stack

- **Frontend**: React 19, Vite 7 (High-Fidelity Glassmorphism UI)
- **Backend**: FastAPI (Asynchronous Python Framework)
- **Data Engine**: Pandas, pdfplumber, PyPDF2
- **Branding**: Night & Emerald Premium Design System

## 📦 Setup & Installation

### Prerequisites
- Python 3.10+
- Node.js 20+

### Backend Setup
1. Navigate to `backend/`
2. Install dependencies: `pip install -r requirements.txt`
3. Launch services: `python main.py`

### Frontend Setup
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Launch development server: `npm run dev`

## 📊 Operational Status
The system is currently configured for local deployment with an active neural extraction pipeline. For enterprise scaling, consider containerization via Docker.
